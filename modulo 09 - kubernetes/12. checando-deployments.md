## Checando Deployments

```
Podemos checar se tudo foi criado corretamente, tanto do deployment quanto a recepção do projeto pelo Pod
```

```
Para verificar o Deployment vamos utilizar: kubectl get deployments
```

```
E para receber mais detalhes deles: kubectl describe deployments
```

```
Desta formar conseguimos saber se o projeto está de fato rodando e também o que está rodando nele
```
