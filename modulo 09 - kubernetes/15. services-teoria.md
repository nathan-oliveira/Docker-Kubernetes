## Services teoria

```
As aplicações do Kubernetes não tem conexão com o mundo externo
```

```
Por isso precisamos criar um service, que é o que possibilita export os Pods
```

```
Isso acontece pois os Pods são criados para serrem destruídos e perderem tudo, ou seja, os dados gerados neles também são apagados
```

```
Então o Service é uma entidade separada dos Pods, que expõe eles a uma rede
```
