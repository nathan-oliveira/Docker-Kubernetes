## Deletar um Deployment

```
Para deletar um Deployment vamos utilizar o comando: kubectl delete deployment <NOME>
```

```
Desta o container não estará mais rodando, pois paramos os Pods
```

```
Assim precisaremos criar um deployment novamente com a mesma ou outra imagem, para acessar algum projeto
```

```
kubectl get deployments
kubectl get pods

kubectl delete deployment flask-deployment

kubectl get deployments
kubectl get pods
```
